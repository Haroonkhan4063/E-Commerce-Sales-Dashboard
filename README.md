# E-Commerce Sales Analysis Dashboard

This project presents an end-to-end **E-Commerce Sales Analysis Dashboard** built using **Python**, focusing on data cleaning, aggregation, and professional data visualization.

The dashboard highlights key business insights such as monthly sales trends, top revenue-generating countries, and best-selling products.

2. **Top 10 Countries by Revenue**
   - Highlights the countries contributing the most to total sales.
   - Useful for geographic performance analysis.

3. **Top 10 Best-Selling Products**
   - Displays products with the highest quantities sold.
   - Supports inventory and product strategy decisions.
